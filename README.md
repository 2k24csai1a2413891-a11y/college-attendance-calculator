# 🎓 PSIT Attendance Calculator

A simple and interactive **Attendance Calculator** designed for college students (PSIT) to calculate attendance percentage using a customized formula.

---

## 🚀 Features

- 📊 Calculates attendance percentage using PSIT formula  
- ✅ Input validation (prevents invalid entries)  
- ⚠️ 75% attendance rule indicator  
- 📉 Suggests:
  - Classes to attend (if below 75%)
  - Classes you can miss (if above 75%)  
- 🎨 Modern UI with:
  - Gradient background  
  - Card layout  
  - Smooth animations  
  - Hover effects  

---

## 🧠 Formula Used
Percentage = 100 - (((Total Absent - Total OAA) / Total Lecture) * 100)

### Where:
- **Total Lecture** → Total classes conducted  
- **Total Absent** → Classes missed  
- **OAA** → Approved leaves (not counted as absence)
